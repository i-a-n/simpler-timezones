to explore that large array one offset at a time, I wrote this tiny function to return all timezones in a given offset:

```js
function getTZs(offset) {
  var retArr = [];
  tzObj.forEach((tz) => {
    if (tz['offset'] === offset) {
      retArr.push(tz['dbname']);
    }
  });
  return retArr;
}
```

now `getTZs(+10:00)` will return all timezones with `UTC+10:00` as their standard-time offset.
